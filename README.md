# 🎬 DeepSeek 短视频脚本生成器

一个基于 **Streamlit + LangChain + DeepSeek** 构建的 AI 短视频脚本生成工具，输入主题、时长和创意度，一键生成爆款标题和结构化脚本。

---

## ✨ 功能介绍
- 🎯 **自动生成标题**：根据视频主题，生成多个吸引人的爆款标题
- 📝 **结构化脚本**：一键生成包含【开头、中间、结尾】的完整短视频脚本
- 🎨 **风格可控**：支持调整脚本创意度，适配不同平台风格
- 🔒 **安全密钥**：密钥仅在本地会话中使用，不会上传到服务器

---

## 🛠️ 技术栈
- **Python 3.8+**
- **Streamlit**：快速搭建交互式 Web 界面
- **LangChain**：构建 LLM 应用的核心框架
- **DeepSeek API**：大模型生成核心能力
